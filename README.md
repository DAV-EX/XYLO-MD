<h2 align="center">ğŸ’» Manual Installation (Developers)</h2>

<details>
<summary><b>Click to expand Installation Instructions</b></summary>
<br>

**1. Prerequisites**
* [Node.js](https://nodejs.org/en/) (v18 or higher)
* [Git](https://git-scm.com/downloads)
* [FFmpeg](https://ffmpeg.org/download.html)

**2. Clone & Install**

```bash
# Clone the repository
git clone [https://github.com/DAV-EX/XYLO-MD](https://github.com/DAV-EX/XYLO-MD)

# Enter directory
cd XYLO-MD

# Install dependencies
npm install
```

3. Configure
Create a .env file or modify config.js:
```
SESSION_ID=YOUR_SESSION_ID_HERE
PREFIX=.
OWNER_NUMBER=2349133354644
MODE=public
```
4. Run

```
npm start
```
